ddDrumr

ddDrumr is drum machine application written in html/css/javascript/jquery. It's developed by the developer community Daladevelop. Originally it was intended to be a contender in the 10k Apart 2011 contest but the fact that it uses audio files meant the 10k limit was quickly passed.
